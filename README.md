# Healthcare Claims Dashboard - Power BI

## Overview

This Power BI dashboard provides insights into healthcare claims data by analyzing claim volume, billed amounts, paid amounts, denial rates, and provider/member performance.

The dashboard helps stakeholders monitor claim processing efficiency and identify trends in payments and denials.

## Key Metrics

- Total Claims: 100
- Total Paid Amount: 255.82K
- Total Billed Amount: 594.98K
- Denial Rate: 17%
- Paid Claim Rate: 63%
- Members With Claims: 59

## Dashboard Features

### Claims Status Analysis
- Paid Claims
- Denied Claims
- Adjusted Claims
- Pending Claims

### Financial Analysis
- Total Billed Amount
- Total Paid Amount
- Year-wise Payment Trends

### Provider Analysis
- Provider-level claim details
- Provider type information
- NPI tracking

### Member Analysis
- Member-wise claim records
- Claim count by member

### Category Analysis
- Endocrine
- Respiratory
- Musculoskeletal
- Cardiovascular
- Dermatology
- Gastrointestinal
- Mental Health
- Oncology
